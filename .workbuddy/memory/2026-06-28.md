## MTP 加速性能测试

**时间**: 2026-06-28 20:20

**任务**: 测试 Qwen3.6-27B MTP 模型的推测解码加速能力

**测试方法**:
- 对比启用/禁用 MTP 的推理速度
- 使用 3 种长度提示词（短/中/长）
- 每个配置生成 512 token

**MTP 配置**:
- `--spec-type draft-mtp`
- `--spec-draft-n-max 2`

**测试结果**:
- 平均加速比: **1.46x** (46% 性能提升)
- 最高加速比: **1.63x** (中长度提示词)
- 无 MTP 速度: 5.7-6.6 tok/s
- 有 MTP 速度: 8.8-9.2 tok/s

**关键发现**:
1. MTP 在 Qwen3.6-27B 上非常有效
2. 中长度提示词加速比最高 (推测命中率高)
3. 当前 `--spec-draft-n-max 2` 可能不是最优，建议测试更高值

- `benchmark_mtp.py` - 测试脚本
- `mtp_benchmark_report.html` - 可视化报告
